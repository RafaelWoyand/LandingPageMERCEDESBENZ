No seguinte arquivo consta imagens e uma landing page direcionada para a divulgação de vendas de carros Mercedes Benz seletos pelo vendedor que obtém o site em mãos.

As opções de Login/Cadastro estão disponíveis ao usuário clicar no ícone superior a esquerda da página web.

No footer consta informações sobre o desenvolvedor como Email, Instagram e o nome completo do mesmo.

Existem links para direcionamento aos canais oficiais Mercedes Benz. Como sites oficiais Mercedes Benz para compra dos automóveis e também para encontrar a Concessionária Mercedes Benz mais perto.
